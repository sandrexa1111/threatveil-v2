# Security Policy

## Passive Scanning Only
ThreatVeil is a passive security decision platform. It relies on public signals, non-intrusive API checks, and user-provided data. It does **not** perform active exploitation, invasive scanning, or unauthorized access attempts.

## Credentials
This repository should never contain real credentials. All API keys and secrets must be managed via environment variables.

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please report it responsibly.

**Do not open a public GitHub issue.**

Please contact the maintainers directly via private channels or email. We are committed to addressing security issues promptly.
